# AICalculatorApp v0.1

Release date: 05/06/2026

This repository stores the source project and the files used for the first AICalculatorApp deployment practice release.

## Expected output

```text
AI Calculator v0.1
2 + 3 = 5
10 - 4 = 6
5 x 6 = 30
20 / 4 = 5
```

## Build command

```bash
mvn clean package
```

## Run command

```bash
java -jar target/ai-calculator-1.0.jar
```
